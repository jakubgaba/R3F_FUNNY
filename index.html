Text Formatting Elements

HTML provides a variety of elements to format text, including headings, paragraphs, and inline text formatting.

Abbreviation Tag Semantic
Used to represent an abbreviation and provides a full description via a title attribute.
<abbr title="Hyper Text Markup Language">HTML</abbr>
HTML
Code Tag Semantic
Used to display code snippets within text.
<p>The <code>&lt;code&gt;</code> tag is used for code snippets</p>

The <code> tag is used for code snippets

Keyboard Tag Semantic
Represents keyboard input or keyboard shortcuts.
<p>Press <kbd>CTRL</kbd> + <kbd>S</kbd> to save a file</p>

Press CTRL + S to save a file

Sample Output Tag Semantic
Represents sample output from a computer program.
<p>The <samp>&lt;samp&gt;</samp> tag for sample output</p>

The <samp> tag for sample output

Variable Tag Semantic
Represents a variable in a mathematical or programming context.
<p>The <var>&lt;var&gt;</var> tag for variables</p>

The <var> tag for variables

Definition Tag Semantic
Represents a definition term.
<p>The <dfn>&lt;dfn&gt;</dfn> tag for definition terms</p>

The <dfn> tag for definition terms

Citation Tag Semantic
Represents a citation or reference to another source.
<p>The <cite>&lt;cite&gt;</cite> tag for citations</p>

The <cite> tag for citations

Modern HTML Elements

Details & Summary HTML5
Creates a disclosure widget to show/hide additional content.
<details>
  <summary>Click to expand</summary>
  <p>Hidden content displayed when expanded</p>
</details>

Dialog Element Modern
Creates a dialog box or modal window.
<button id="showDialog">Open Dialog</button>
<dialog id="myDialog">
  <h3>Dialog Title</h3>
  <p>This is a dialog window.</p>
  <button id="closeDialog">Close</button>
</dialog>

Progress Element HTML5
Represents the completion progress of a task.
<label for="task-progress">Task progress:</label>
<progress id="task-progress" value="70" max="100">70%</progress>
70%
Meter Element HTML5
Represents a scalar measurement within a known range.
<label for="disk-usage">Disk usage:</label>
<meter id="disk-usage" value="0.7" min="0" max="1" low="0.3" high="0.8" optimum="0.5">70%</meter>
70%
Picture Element Responsive
Container for multiple image sources for responsive images.
<picture>
  <source media="(min-width: 800px)" srcset="large-image.jpg">
  <source media="(min-width: 480px)" srcset="medium-image.jpg">
  <img src="small-image.jpg" alt="Responsive image">
</picture>

The picture element allows developers to provide different images based on device characteristics like screen size, resolution, or orientation.

Interactive Elements

Datalist Element Forms
Provides a list of predefined options for input elements.
<label for="browser">Choose your browser:</label>
<input list="browsers" id="browser" name="browser">
<datalist id="browsers">
  <option value="Chrome">
  <option value="Firefox">
  <option value="Safari">
  <option value="Edge">
</datalist>
Output Element Forms
Represents the result of a calculation or user action.
<form oninput="result.value=parseInt(a.value)+parseInt(b.value)">
  <input type="range" id="a" value="50"> +
  <input type="number" id="b" value="50"> =
  <output name="result" for="a b">100</output>
</form>
+ = 100

A

Advanced Features

Template Element Modern
Contains HTML that is not rendered initially but can be used with JavaScript. 
<template id="userTemplate">
  <div class="user-card">
    <h4>User Name</h4>
    <p>Email: user@example.com</p>
  </div>
</template>
Template content is not visible until cloned with JavaScript.
Content Editable Attribute Interactive
Makes any element editable by the user.
<div contenteditable="true" style="padding:10px;border:1px dashed #ccc;">
  This content is editable. Click to edit.
</div>
This content is editable. Click to edit.
Lazy Loading Performance
Defers loading images until they're near the viewport.
<img src="large-image.jpg" loading="lazy" alt="Lazy loaded image">

The loading="lazy" attribute tells browsers to defer loading off-screen images until users scroll near them, improving page load performance.

Color Input Type Forms
Provides a color picker interface.
<label for="colorpicker">Select a color:</label>
<input type="color" id="colorpicker" value="#3498db">
Slot Element Web Components
Placeholder inside a web component used in Shadow DOM.
<template id="element-template">
  <h2><slot name="title">Default Title</slot></h2>
  <div><slot name="content">Default content</slot></div>
</template>

Slots allow custom elements to be composed with custom content, enabling reusable components with replaceable parts.

Emerging HTML Features

Dialog with Modal Attribute Experimental
Creates a modal dialog that disables interaction with the rest of the page. 
<button id="showModal">Show Modal Dialog</button>
<dialog id="modalDialog">
  <h3>Modal Dialog</h3>
  <p>This is a modal dialog with a backdrop.</p>
  <form method="dialog">
    <button>Close</button>
  </form>
</dialog>

Modal Dialog

This is a modal dialog with a backdrop.
Popover API Emerging
Creates a popover element that can be toggled without custom JavaScript. 
<button popovertarget="myPopover">Show Popover</button>
<div id="myPopover" popover>
  <h3>Popover Content</h3>
  <p>This is popover content that appears on demand.</p>
</div>

The Popover API allows creating popover UIs (tooltips, menus, dialogs) that are properly layered with good keyboard navigation, without requiring custom JavaScript.

Inert Attribute Emerging
Makes elements unfocusable and unselectable, useful for modal overlays.
<div inert>
  <button>This button cannot be focused or clicked</button>
  <input type="text" placeholder="This input is disabled">
</div>

The inert attribute makes an element and all its contents unfocusable and unavailable for interaction, which is useful for maintaining focus inside modal dialogs.
